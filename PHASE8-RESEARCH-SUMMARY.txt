2. QWEN MODEL VARIANTS
   - 30B: Tested, tool sequencing 70% reliable
   - 35B: Available locally, untested (17% more params)
   - 70B: Available locally, untested (2.3x params)
   
   HYPOTHESIS: Scaling effect overcomes tool-sequencing ceiling

3. BEST-PRACTICES PATTERNS
   - Explicit negative instructions: +8-15% effective
   - Tool sequencing constraints: 0% improvement (model limit)
   - Few-shot examples: untested but theory sound
   - Temperature tuning: untested but medium likelihood

4. ISSUE #475 INSIGHTS
   - Model forgets opening <tool_call> tag
   - "Do NOT omit" reduces 15-20% error to 2-5%
   - NEGATIVE instructions > POSITIVE instructions

5. PHASE 7 ROOT CAUSE
   v2.0 (tojson + enhanced prompting) still fails tool sequencing
   => Problem is MODEL CAPABILITY, not TEMPLATE
